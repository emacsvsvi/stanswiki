This knowledge base should include your chats, lists of things (tasks, movies to see, books to read, etc.), documentation for code projects, etc. Also links to run programs ai and I have written. AI should help to organize this and create links between things when it makes sense. It should also have a log of all the updates it makes so that a user can understand the changes. Finally, it should ask the user where to put stuff if it isn't sure.
Using linux or windows create a system that starts with AI installed. The AI controls all of the installation of applications, uses the wiki in the above project to control all information. You should easily be able to use voice or the keyboard to control the AI and therefore the PC, it's applications and knowledge base
