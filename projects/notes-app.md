Application to take text in an image and generate a text file of the data in the image. Send an email to [stan@stanparker.com](mailto:stan@stanparker.ccom) with the subject "notes123"" then this app will take the attached image and generate a text file. The text file gets saved to [Notes Directory](file:///C:/mydata/projects/notes-skill/notes)
